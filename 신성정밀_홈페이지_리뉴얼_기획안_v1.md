## 6. 페이지별 UX / 인터랙션 기획 (시각적 변화 & 모션 메커니즘)

### 6-1. 메인 히어로
첫 화면은 풀스크린 다크 네이비 배경에 Three.js(또는 Spline) 기반의 메탈릭 정밀 부품(캔 케이스 또는 정밀 기어 형상) 와이어프레임이 천천히 자체 회전합니다. 마우스를 움직이면 오브젝트가 시차(parallax)를 두고 미세하게 따라오며 "떠 있는" 느낌을 줍니다. 스크롤을 시작하면 GSAP ScrollTrigger로 헤드라인 텍스트가 한 줄씩 블러(blur) 상태에서 포커스로 전환되며 올라오고, 배경 그라디언트가 네이비→차콜로 서서히 전환됩니다. 헤드라인 옆에는 핵심 지표(예: 분당 생산량, 소재 절감률)가 0에서 목표값까지 카운트업되는 숫자 애니메이션이 동반됩니다. CTA 버튼에는 마우스가 다가가면 버튼이 자석처럼 끌려오는 **Magnetic Hover**와, 버튼 테두리를 따라 네온 라인이 흐르는 **Border Flow** 효과를 적용합니다.

### 6-2. 가로 스크롤 설비 소개 섹션
세로 스크롤 중 특정 구간에 진입하면 GSAP ScrollTrigger로 스크롤을 가로 캐러셀로 전환(scroll-hijack)하여, 휠 동작이 제품 카드를 옆으로 슬라이드시키는 데 사용됩니다. 각 카드는 반투명 블러 + 실버 보더의 글래스모피즘 패널로 구성되며, 호버 시 카드가 3D perspective로 살짝 기울고(tilt) Z축으로 떠오르며 사양 정보가 페이드인됩니다. 하단에는 스크롤 진행률에 따라 네온 블루로 채워지는 가로 프로그레스 바를 배치합니다.

### 6-3. 기술력 섹션
특허·인증 마크는 그리드로 배치하고, 스크롤 진입 시 좌→우 또는 하→상으로 순차(stagger) 페이드업됩니다. 생산성 수치(분당 1,000개, 소재 절감 14.6% 등)는 원형 게이지 또는 라인 그래프가 스크롤 트리거 시점에 0에서 목표치까지 차오르는 애니메이션으로 표현합니다. 배경에는 정밀 도면(blueprint) 느낌의 그리드 라인이 매우 느리게 흐르는 루핑 애니메이션을 깔아 "기술 문서" 같은 분위기를 강조합니다.

### 6-4. 적용 분야 섹션
산업별(전기차/식음료/화학/조선 등) 탭을 클릭하면 콘텐츠가 글래스 패널 형태로 크로스페이드 전환되고, 탭 인디케이터(언더라인)가 부드럽게 슬라이드 이동합니다. 각 산업 이미지에는 약한 패럴랙스와 다크 그라디언트 오버레이를 적용해 텍스트 가독성을 확보합니다.

### 6-5. 고객지원 / 문의
입력 필드에 포커스하면 네온 블루 글로우 보더가 나타나고, 실시간 유효성 검사 아이콘이 체크 애니메이션으로 표시됩니다. "오시는 길" 지도는 다크모드 커스텀 스킨(카카오맵/구글맵 다크 테마)을 적용해 전체 톤앤매너와 통일감을 줍니다.

## 8. 기술 스택 및 아키텍처

- **프론트엔드**: Vite + Tailwind CSS + GSAP(ScrollTrigger) + Three.js/Spline — 섹션별(메인 히어로, 가로스크롤 설비소개 등) 모듈화된 컴포넌트 단위로 개발
- **백엔드/운영 솔루션**: 그누보드5(PHP) 기반 자체 개발 페이지(관리자 포함)
  - 프론트엔드에서 완성한 정적 HTML/CSS/JS 결과물을 그누보드 **스킨(skin) 테마**로 포팅하는 2단계 작업이 필요합니다.
  - 공지사항/자료실/문의 게시판은 그누보드 기본 게시판 기능을 활용하고, 디자인만 커스텀 스킨으로 덮어씁니다.
  - 문의/기술상담 폼은 그누보드 게시판 연동 또는 별도 메일 발송 연동 중 선택이 필요합니다.
- **자산 확인 필요**: 실제 설비 구동 영상, 3D 모델(또는 3D 스캔용 실물 촬영) 보유 여부 — 인터랙션 품질에 직접 영향
